This is the quick to learning the archive set up screen.  Please see the detailed help screen whenever you have specific questions.

 

After you have identified everything to archive, expect the archiving function to run for a long time the first time.  You will see a status bar at the top.  You may use your computer for other uses when it is archiving.  We recommend you always close Outlook when archiving Outlook emails so no emails are skipped.

Archiving Documents:

The section of the screen highlighted below is the content/documents archive set up area:

 

Steps

1.       After pulling down a retention rule (e.g. save for 10 years), click on “select dir for archive” just below it.   It will open up a computer browser to set directories.

2.       On the very right notice the listing of file types.  Highlight the file types to archive and click on the“include” button that will appear.   As an alternative,  pull down “archive profile groupings” like “Office documents” and click apply.  Your files will be now listed in the “included column”.

3.       Notice the area that lists archive options:  subdirectories,  version files (so a new archive doesn’t write over old files), OCR (object character recognition on graphic files like JPEGs) and capture Metadata (Microsoft Office extra file attributes).  Select the options you want.

4.       Save Changes.

5.       Click on “Archive Content”.    The first archive will run for awhile.

Setting up Automatic Archiving for Documents/ Email:

To automatically archive any changes or additions going forward,  go to the top middle “Polling Setup”.  We recommend “Load at Start up” and choose a daily time under “Run Archiver”.  Make sure it’s at a time when your PC is on (not in sleep mode as it won’t archive).  Archiving will run in the background.

Archiving Outlook Emails:

The section of the screen highlighted below shows where you identify the content to be archived:

 

Steps

1.        After pulling down the desired set of folder groups (e.g. Personal Folders) click on “Avail for Archive”.

2.       The yellow grid will populate.  Highlight those folders that you want archived.  (Note that calendar and contacts will not be archived here even if you select it).

3.       Select the boxes how you want these emails to archive.  You must select “Archive Emails in folder” in order to archive.

4.       Click on “Activate”.

5.       Then Click on “Archive Outlook Emails”.  We recommend you always close Outlook when archiving Outlook emails.  Note:  You may experience a permissions request behind your window from Outlook depending on your computer set up.  It needs to be selected prior to archive.

 

Scheduling

 After setting up all the content you want archived, your first archive will take some time.  Set your computer through the control panele so it will not go to sleep before the archive is complete.  Then select either “Archive ALL” or do emails and content separately. 

To automatically set up archives going forward, select “Schedule”.  A new screen will appear that interfaces with Microsoft Scheduler.  For advanced settings that can be used in Microsoft Scheduler, please see the following link:
