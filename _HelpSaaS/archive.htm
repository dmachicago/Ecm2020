This is the quick to learning the archive set up screen.  Please see the detailed help screen whenever you have specific questions.

 

After you have identified everything to archive, expect the archiving function to run for a long time the first time.  You will see a status bar at the top.  You may use your computer for other uses when it is archiving.  We recommend you always close Outlook when archiving Outlook emails so no emails are skipped.

Archiving Documents:

The section of the screen highlighted below is the content/documents archive set up area:

 

Steps:

1.       Pull down a retention rule for the directory path you are about to use (e.g. save for 10 years - these rules are set up by your administrator).    Click on “Select Dir for Archive” just below it.   It will open up a computer browser for you to then select directories.  Select your directory path.  ECM Library can archive sub-folders for this path.

2.       On the right of the screen notice a listing of file types.  Highlight the file types to archive and then click on the “Include” button that appears. Your files will be now listed in the “Included” column.  (As an alternative shortcut, click on the arrow below the “Select Filetype Profile” and select groupings of similar file types like “Office Documents” and “Apply”.  )

3.       Find the area that lists archive options:  Include Subdirectories,  Version Files (so a new archive doesn’t write over old files), OCR (object character recognition on graphic files like JPEGs) and capture Metadata (Microsoft Office extra file attributes).  Select the options you want.

4.       Click on “Save Changes to Archive”

5.       If you’re ready to archive, go to the Archive Menu and select Content to archive immediately, or go to Schedule to set up an automatic schedule.

Archiving Outlook Emails:

The section of the screen highlighted below shows where you identify the content to be archived:

 

Steps

1.       A. Pull down and select the desired set of folder groups with the arrow ( on screen beside test).  For example, click on Personal Folders.

2.       Click on “Avail for Archive”. 

3.       The yellow grid will populate.  Highlight those folders that you want archived.  (Note that calendar and contacts will not be archived here even if you select it).

4.       Check those boxes below the yellow grid to select how you want these emails to archive.  You must select “Archive Emails in folder” in order to archive.

5.       Choose the number of “Retention Years” for all those folders you have selected.

6.       Click on “Activate”.

7.       If you are ready to just archive emails,  pull down under the “Archive” top Menu “Outlook”.  Otherwise choose “Schedule”.  We recommend you always close Outlook when archiving Outlook emails.  Note:  You may experience a permissions request behind your window from Outlook depending on your computer set up.  It needs to be selected prior to archive.

 Under Archive Menu: Setting up Scheduling of Automatic Future Archives:

To automatically archive any changes or additions to content and emails going forward, go under the “Menu Archive” and select “Schedule”.  This will bring up the separate Archive application.  This application takes advantage of Microsoft Scheduler, and makes archiving easy, whether or not your application is running at the time or not.   To set up your schedule, click on what you want to have archived (All, file folders, Outlook, Exchange), add in when to archive (e.g. Daily), time to run and start time.  Finally, click “Add New Selected Task” and it will the new task will appear in the grid above.   Note that the specific email folders, directory paths and/or exchange accounts need to be set up in the Archive Set Up screen, with all their archive parameters in order for anything to be identified during Archive.

Finally, highlight your Task Name, and click on “Enable Archive” to turn on the Scheduled Task.  You can “Disable Archive” by highlighting the Task name.  This capability allows you to choose to turn the automatic schedule on or off by task.

 

For advanced settings that can be used in Microsoft Scheduler, please see the following link:
